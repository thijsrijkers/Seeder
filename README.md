# Seeder
Seeds generator that uses specific biome and coordinates

## Building
To build Seeder you need:
- A recent version of Intellij
- Gradle (If needed)
<h2> Gradle installation</h2>
<pre>
  <code>
    $ java -version
    java version "1.8.0_121"
  </code>
  </pre>
  <pre>
  <code>
    $ sdk install gradle 6.6.1
  </code>
</pre>
